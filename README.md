# Video Meeting

### Features
- Is 100% free and open-source
- No account needed
- Unlimited users
- Messaging chat and video streaming in real-time
- Screen sharing to present documents, slides, and more
- Everyting is peer-to-peer thanks to webrtc


### Local setup

1. `yarn install`
2. `yarn dev`
